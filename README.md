# Dotfiles

Here you can find my configuration files (usually called dotfiles) for a number
of things. Below I'll give a bit more detail on what you can find here, but the
bits that I used the most are these:

- VSCode stuff, which you can find in the `vscode` folder, including
  keybindings, settings, extensions, snippets and tasks
- Vim stuff, like settings, mappings and plugins. See the `vim` folder
- `.tmux.conf`, essential for anyone using tmux. If you are not, I strongly
  encourage you to try it out
- `.rc.scratch`, which is a simple script that I bind to my <kbd>insert</kbd>
  key to be able to launch (and hide) a new tmux session just pressing that key
- `.gitconfig`, of course
- `deps.edn`, with some useful Clojure stuff

There's also a bunch of other stuff, from Sublime Text settings I used to use in
the past to the desktop entry for the apps I installed using AppImage, and more.

Feel free to take a look!
